# ATM Interface

A Java-based ATM simulation project that allows users to perform banking operations such as deposit, withdrawal, and balance inquiry using Object-Oriented Programming (OOP) concepts.

## Features
- Deposit Money
- Withdraw Money
- Check Account Balance
- Input Validation
- User-Friendly ATM Menu

## Technologies Used
- Java
- OOP Concepts
- Scanner Class
- Loops
- Switch Statements
